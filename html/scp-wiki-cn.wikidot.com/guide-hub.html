指导中心

欢迎来到指导中心！如果你是在寻找答案或者意见，你应该能在这里找到它们。如果你有写作一篇指导的想法，想要添加到这个页面，请和高级人员联系，给他们看你的草稿。

该页也包括中文维基的注意事项。

请在继续前阅读

  • 站规：你应该知道并且完全理解wiki的规则。不了解规则并不成其为你打破它的理由。在继续阅读任何其他东西前，请无比阅读这个。

首要阅读

  • 给初心者的指南 ：这篇指导包括了站点申请，总体站点行为守则，成为一个好成员的几点建议，写作的建议以及高级人员列表。
  • Chat Guide ：基金会实时聊天室的指导。包括了注册，连接，会被禁止的冒犯性言语，聊天室管理员，讨论建议以及从属的聊天室。
  • Criticism Policy：包含了SCP Wiki关于提出和接受批评的政策。
  • FAQ：经常被问及的问题。包含关于SCP宇宙的信息，站点如何运作以及适当的表现的问题。
  • 授权指南：有用的信息，对于相对多产的成员更是十分重要。对于那些想要创作衍生作品的人来说，这是一个必要且实用的指导。这些衍生物包括基于SCP基金会的游戏，海报以及商品。它解释了本站点中知识共享（Creative Commons）条例的使用细则和必要性。

中文WIKI相关

这里的指导都是由中国分部的人员撰写，能为于中文维基的工作带来帮助。(未完成)

  • 维基语法：翻译自Wikidot的语法指南，列出所有在这里会用到的语法。
  • 已翻译文档搬运指南：指出自其他地方搬运一篇已翻译文档到维基时的基本流程以及注意事项。
  • 斯卡麗的完美玩梗教室：不想好不容易找到一個不錯的梗結果被罵「玩梗自重」的話，最好來看一看

写作指导

  • 如何撰写一篇SCP文档：如标题所述。包括想法的发展形成，收容，语调&检查制度，人型生物，格式以及模板。在写一篇文章之前，请阅读这个。
  • Image Use Policy：如果你要在你的SCP项目或者基金会故事里用上图片的话，看看这个。
  • 标签指导 ：标签指导是给SCP项目，故事和其他站点页面打标签的指南。它包含了官方标签的完整清单和详细定义，以及一些特别情况的例证，告诉你这时候打上哪个标签比较合适以及这些标签是什么。
  • Deletions Guide ：解释了删除页面方面的政策，包括了你要知道的所有内容：如何、何时以及具体操作过程。请在发表文章之前阅读此文。
  • Technical Words：各种科学和工程学术语以它们的含义。对于让你的作品听起来更加专业科学十分有用。
  • Expert Witnesses ：用户能在这里找到一些具体信息以及它们的资格证书。如果你正在写一篇需要专业知识的文章，可以考虑这里。

各种有用的信息与文章

  • 关于SCP基金会：在基金会的宇宙框架内为你从最基本的层次讲解基金会是什么。
  • 经验规则：一系列如何让人点赞的规则，从站点内经验丰富的成员处收集整理而成。
  • Guide to the Shit You Likely Missed：正如标题所述。一个简短的清单，列举了许多新人容易忽视的要点。
  • Classified and You：一份简明扼要的报告，关于机密的含义以及如何接触机密文件材料。
  • 襌與資料刪除技術 ：对于删节、数据删除和检查制度的实用指导。
  • Doing the Safety Dance ：关于如何写作合理而有趣的收容措施的指导。
  • The Foundation and Evil：基金会不残忍但冷漠。
  • Conservation of WTF，或者“为什么兔子需要两个脑子？”： 一篇文章，大致说明了写作涉及异常时应该采取的精神状态。多怪就过了头？
  • Understanding Memetics：一篇解释SCP基金会宇宙观下模因到底是什么东西。
  • Let’s Talk About Character For A Minute：一篇文章，解释基金会宇宙中超自然特工与博士的主要问题，对总体实现角色完整愿望的简短讨论，以及如何在你自己的写作中避免常见的误区。
  • Random But Useful：简短的介绍，涵盖了内容广泛的零碎要点。
  • So You Want To Write A Humanoid SCP Object：这篇文章旨在为那些写作人形SCP项目的人提供方法建议。
  • 如何吓人于无形：如标题所述。元音省略以及五个关键短语。
  • Technical Writing ：给那些并非科学技术人员的技术性介绍。我们中大多数是写手而并非科学家。当你想要度过写作中的“技术性”难关的时候，这可能很有帮助。
  • Hornby’s Audio Guide：关于如何制作一篇音频戏剧的十分透彻的指导。
  • 解密：何为客观超然的语气： 助你在條目裡掌握那難以捉摸的客觀語氣。

幽默的文章

  • The Big List of Overdone SCP Cliches ：这个清单里的东西要么被用烂了要么不适合基金会的风格。该文章语言诙谐幽默，不过自然并非死规定。如果有人指着这个页面告诉你 你的SCP烂透了又不加以解释，你大可以针锋相对顶回去。
  • Cliches and You: An Educational Film：对上方所述的陈词滥调的拓展清单，并清楚阐明了每一行的意思。
  • 作家观念：一个半开玩笑的清单，讽刺那些新手经常会有的老套观念。
  • Standard Form For Downvotes

Aelanna给聪明人的话

Aelanna十分热心地创作了一系列非官方文章与指导，就写作与批评从多角度进行了讨论。

  • Common SCP Pitfalls：对糟糕的SCP文章中反复出现的问题的粗略概括。
  • Documentation Tips：如何以冷静的SCP式风格写作一个故事。在构建你的文章时应该考虑什么。
  • 術語大全：你在这里会见到的wiki行话汇总表。
